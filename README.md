# HR Analytics: Employee Attrition Analysis

## Overview

This project analyzes employee attrition data to identify factors that influence employee turnover and support data-driven HR decision-making.

## Database Source

https://www.kaggle.com/datasets/giripujar/hr-analytics

## Objectives

* Explore employee demographics and workplace characteristics
* Identify patterns related to employee attrition
* Perform data cleaning and exploratory data analysis (EDA)
* Generate insights to help improve employee retention

## Key Analysis

* Data preprocessing and cleaning
* Employee attrition distribution analysis
* Correlation and feature analysis
* Visualization of workforce trends
* HR business insights and recommendations

## Results

The analysis highlights key factors associated with employee turnover and provides actionable recommendations for improving employee retention strategies.
